C. Tsundoku

因为只能顺序选择两个书架里的书，所以定义前缀数组\(a_{k} = \sum_{i=1}^{k}A_{i}\)，\(b_{k} = \sum_{i=1}^{k}B_{i}\)。 题意归结为:对于满足条件\(a_{i} + b_{j}≤K\)的整数 \(i, j (0 ≤ i ≤ N, 0 ≤ j ≤ M)\)，找到\(i+j\)的最大值。

思路：从前到后扫描\(i\)，在每一个\(i\)下找到满足\(b_{j} \leq K-a_{i}\)的最大整数\(j\)，不难发现\(j\)是递减的，也即\(i\)是从0到N，\(j\)是从M到0。

D.Sum of Divisors

题意: 对于任意整数\(x\)，令\(f(x)\)表示其正因子个数。输入整数N，求\(\sum_{k=1}^{N} k \times f(k)\).

E - NEQ

题意：考虑长度为N的序列对\(A_{1}...A_{N}和B_{1}...B_{N}\)，其中所有的数都在\(1\)到\(M\)之间(\(N \leq M\))，求满足以下条件的序列对数量(mod 10^9+7)
(1) 对于任意 \(1 \leq i \leq N\)，\(A_{i} \neq B_{i}\)。
(2) 对于任意\(1 \leq i < j \leq N\)，\(A_{i} \neq A_{j}\)并且\(B_{i} \neq B_{j}\)。

思路：在条件(2)下，令\(S\)表示所有序列对构成的集合，\(T_{k}\)表示满足条件\(A_{k} = B_{k}\)的序列对集合，则题意所求为\(|S/\bigcup_{k=1}^{N}T_{k}|\)，根据容斥原理，\(|S/\bigcup_{k=1}^{N}T_{k}|=|S| + (-1)^{1} \sum_{k=1}^{N}|T_{k}| + (-1)^{2} \sum_{k=1}^{N}\sum_{h=1,h \neq k}^{N}|T_{k} \bigcap T_{h}|...\)， 不难求出其中第\(i\)项为\((-1)^{i}C_{N}^{i} \times \tbinom{M}{i} \times (\tbinom{M-i}{N-i})^2\)，故最终结果为\(\sum_{i=0}^{N}(-1)^{i} \times C_{N}^{i} \times \tbinom{M}{i} \times (\tbinom{M-i}{N-i})^2\)。
